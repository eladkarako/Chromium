look for switch.cc

it will actually find *switch*\.*cc* (loose find),  
you can create a new list to download/concat. 
keep old versions since some of the old switches are still working.

then run:
--------------------------------------------------------------------------
NodeList.prototype.map = Array.prototype.map;       //polyfill for NodeList.
document.querySelectorAll('a.tree-browser-result')
        .map(function(e){                           //HREF modifed into "RAW" mode (plain text)
          return e.href
                  .replace(/\/github.com\//i,"\/raw.githubusercontent.com\/")
                  .replace(/\/blob\//i,"\/")
        })
        .sort()
        .join('\r\n');
--------------------------------------------------------------------------

the code above query all the <A> elements, 
extracts just the HREF attribute (".href" calculates a fully-qualified URL), 
modifies the URL to point to the "RAW" (plain text) version of the file, 
for example:
    https://github.com/chromium/chromium/blob/master/weblayer/public/common/switches.cc
  is changed into 
    https://raw.githubusercontent.com/chromium/chromium/master/weblayer/public/common/switches.cc
